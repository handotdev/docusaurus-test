---
title: "fern add"
description: "Add a code generator to `generators.yml`."
---

## Usage

```
fern add <generator-name>
```

## Examples

Add the TypeScript SDK generator:

```
fern add typescript
```

Add the Postman collection generator:

```
fern add postman
```
