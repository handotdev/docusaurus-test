---
title: "Code Generators"
description: "Fern makes it easy to code generate SDKs, server interfaces, a Postman
collection, and an OpenAPI description."
---

### Generate SDKs

#### JavaScript/TypeScript

```
fernapi/fern-typescript-sdk
```

A JavaScript SDK, fully typed with TypeScript declarations. Build Node.js, web,
and mobile web applications.

#### Java

```
fernapi/fern-java-sdk
```

A Java SDK.

#### Python

```
fernapi/fern-python-sdk
```

A Python SDK.

### Generate server interfaces

#### TypeScript Express

```
fernapi/fern-typescript-express
```

Validation that your JavaScript or TypeScript Express server correctly
implements your API.

<Warning>Requires private beta login!</Warning>

#### Java Spring

```
fernapi/fern-java-spring
```

Validation that your Java Spring server correctly implements your API.

<Warning>Requires private beta login!</Warning>

#### Python FastAPI

```
fernapi/fern-python-fastapi
```

Validation that your Python FastAPI server correctly implements your API.

<Warning>Requires private beta login!</Warning>

### Generate a Postman Collection

```
fernapi/fern-postman
```

Generates a [Postman Collection](https://www.postman.com/collection). If you'd
like to use the Postman integration to auto-update your collection,
[read on](/features/postman).

### Generate an OpenAPI description

```
fernapi/fern-openapi
```

Generates an [OpenAPI description](https://swagger.io/resources/open-api/).
