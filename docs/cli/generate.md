---
title: fern generate
---

Run the draft tasks from **generators.yml**.

## Usage

<!-- markdownlint-disable MD040 -->

```
fern generate
```

## Example

If we have previously ran `fern add typescript` and now run `fern generate`:

```
┌─
│ ✓  fernapi/fern-typescript-sdk
│    ◦ @fern-examples/client@0.0.1
└─
```
